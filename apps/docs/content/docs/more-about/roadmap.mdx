We soft-launched https://mcpay.tech in 25th August 2025.

## October-November 2025

- MCP monetization wrapper (mcp2.mcpay.tech). 
    - Uses the `mcpay` SDK and a proxy-like system to monetize and index servers. Similar to current `/register` route in the app, but more modular.
- Proxy system with hooks
    - Uses OAuth 
- SDK add-ons
    - Support for monetizing other resources than tools.
- Improved indexing system. No static tool discovery.
